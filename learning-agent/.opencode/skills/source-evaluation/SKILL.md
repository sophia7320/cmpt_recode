# 信息真伪评估框架

## 核心理念

互联网上没有"天然可信"的信息。每一条信息都需要先通过评估筛子，才能作为学习依据。

## 方法一：CRAAP 评估法

**来源**：加州州立大学奇科分校 (2010)

对每条信息按以下 5 个维度打分（每项 1-5 分）：

| 维度 | 含义 | 核心问题 |
|------|------|----------|
| **Currency** 时效性 | 信息是否更新？ | 发布/更新时间？该领域知识更新速度？ |
| **Relevance** 相关性 | 与你的需求相关吗？ | 直接回答你的问题吗？适合你的水平吗？ |
| **Authority** 权威性 | 来源靠谱吗？ | 作者/机构是什么背景？有相关资质吗？ |
| **Accuracy** 准确性 | 内容正确吗？ | 有证据/数据支持吗？能交叉验证吗？ |
| **Purpose** 目的性 | 为什么发布？ | 是教育目的还是商业推广？有无偏见？ |

评级标准：
- 总分 21-25：高可信度（可作为学习依据）
- 总分 15-20：中可信度（可参考但需交叉验证）
- 总分 10-14：低可信度（不推荐使用）
- 总分 < 10：不可信（抵制使用）

## 方法二：SIFT 四步法

**来源**：Mike Caulfield (2017)

### S - Stop（暂停）
在阅读任何信息之前，先问自己：
- 我知道这个网站/来源吗？
- 它的声誉如何？

### I - Investigate the source（调查来源）
不要在这个页面内找答案。打开新标签页搜索：
- 搜索作者姓名 + 机构
- 搜索网站名称 + "可信度" 或 "评价"
- 查看 Wikipedia 对该来源的描述

### F - Find better coverage（找更权威的覆盖）
不满足于一个来源。搜索同一主题，看更权威的媒体/机构怎么说。

### T - Trace claims to original（追溯原始出处）
- 如果文章引用了一项研究，找原始论文
- 如果引用了一个统计数字，找原始报告
- 大多数引用在传播中出现歪曲或断章取义

## 常见逻辑谬误识别清单

| 谬误 | 描述 | 示例 |
|------|------|------|
| 诉诸权威 | 用名人/权威代替论证 | "爱因斯坦也信这个" |
| 稻草人 | 歪曲对方观点再攻击 | "你说教育重要，意思是不用工作了？" |
| 假两难 | 只给两个选项，忽略其他 | "要么支持A，要么就是B的同党" |
| 滑坡谬误 | 无充分依据地夸大后果链 | "今天不背单词，明天挂科，后天退学" |
| 循环论证 | 用结论证明前提 | "这东西有效，因为它能起作用" |
| 因果混淆 | 相关≠因果 | "冰激凌销量和溺水人数都上升→冰激凌导致溺水" |
| 以偏概全 | 从个别案例推出一般结论 | "我一个朋友...所以所有人..." |
| 诉诸情感 | 用情感操控替代逻辑 | "想想那些可怜的孩子"（但无关论点） |
| 人身攻击 | 攻击发言者而非论据 | "你学历不高所以你说的不对" |
| 红鲱鱼 | 引入无关话题转移焦点 | 讨论A时突然扯到B |
| 诉诸传统 | 因为一直这样所以应该这样 | "我们从来都是这么做的" |
| 乐队花车 | 因为大家都信所以正确 | "这么多人关注，肯定有道理" |

## 学术共识判断

| 判断方法 | 说明 |
|----------|------|
| 区分个别论文与学科共识 | 一篇论文≠学界共识。寻找综述文章(Review)或元分析(Meta-analysis) |
| 警惕掠夺性期刊 | 出版速度快+高额版面费+无同行评审的期刊不可信 |
| 检查引用情况 | 被引量高≠一定正确，但被引量极低通常说明未被广泛接受 |
| 观察学科内争议 | 如果一个话题在学界有争议，用"部分研究认为...；另有研究指出..."而非确定语气 |

## 输出格式

评估结果包含可信度标记：

```
可靠性评级：⭐⭐⭐ 高 / ⭐⭐ 中 / ⭐ 低
评估方法：CRAAP
关键依据：
  - 权威性：{说明}
  - 准确性：{是否有交叉验证}
  - 时效性：{发布时间}
注意事项：{如有偏见或其他问题}
```

## 反模式

- 只看第一条搜索结果
- 只在一个来源内部验证其可信度
- 只看域名判断（.org 不代表非盈利可信，.edu 可能是学生博客）
- 看到"研究表明"就不管具体引用了什么研究
