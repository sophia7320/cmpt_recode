# 费曼学习法 (Feynman Technique)

**来源**：诺贝尔物理学奖得主 Richard Feynman 的教学方法。

## 核心理念

如果你不能简单地解释一件事，说明你还没有真正理解它。

## 四步流程

### 步骤 1：选定概念
确定要讲解的概念，明确它的边界——这个概念解释什么、不解释什么。

### 步骤 2：向孩子解释
想象你在向一个没有该领域背景的 12 岁孩子讲解。要求：
- 不使用任何专业术语（除非先定义它）
- 每个句子不超过 25 个字
- 使用日常生活中的例子

### 步骤 3：识别卡顿
当你发现自己：
- 用"大概"、"基本上是"等模糊词
- 跳过某个细节
- 解释不下去

这些地方就是你的理解盲区。回到学习材料，重新学习这部分。

### 步骤 4：简化与类比
最终版本应该：
- 用一句话提炼核心（"这个概念其实就是在说..."）
- 给出一个生活化类比（"就像..."）
- 如果能用图形表达，用文字描述图形结构

## 在教学中应用

当你讲解任何概念时，遵循这个输出结构：

```
{概念名称}

一句话版：{用一句话说清这个概念}

类比版：想象一下，{生活化类比}

拆解版：
1. 首先，{最基础的部分}
2. 然后，{在此基础上}  
3. 最后，{完整的理解}

举个栗子：{具体例子}

需要注意：{常见误解} ← 很多人以为X，其实不对，因为...
```

## 检查清单

每次讲解后自查：
- [ ] 可以用一句话说清吗？
- [ ] 用了类比吗？
- [ ] 指出并纠正了常见误解吗？
- [ ] 用户能用自己的话复述吗？

## 反模式

- 使用比要解释的概念更复杂的术语来解释
- 跳过类比，只用抽象描述
- 默认用户已经理解了前置知识
- 一次性塞入过多概念
