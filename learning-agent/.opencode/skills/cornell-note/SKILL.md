# 康奈尔笔记法 (Cornell Note-Taking System)

**来源**：Walter Pauk，康奈尔大学 (Cornell University)

## 核心理念

把笔记分三块：记录 + 提炼 + 总结。让笔记从"写下来"升级为"消化掉"。

## 三栏结构

```
┌─────────────────────────────────────┐
│ 标题：{知识点}         日期：{date}  │
├──────────────┬──────────────────────┤
│ 线索栏        │ 笔记栏                │
│ (关键词/问题) │ (记录内容)            │
│              │                      │
│ · 核心概念是?  │ {用自己话概括的核心点}  │
│              │                      │
│ · 关键步骤?   │ {步骤1}              │
│              │ {步骤2}              │
│              │                      │
│ · 与XX的区别?  │ {对比表格}            │
│              │                      │
│ · 要注意什么?  │ {注意事项}            │
├──────────────┴──────────────────────┤
│ 总结栏                                │
│ {用2-3句话概括整篇笔记的核心内容}        │
└─────────────────────────────────────┘
```

## 在整理笔记时应用

输出结构：

```markdown
# {知识点名称}
- 日期：{date}
- 科目：{subject}

## 一句话总结
{30字以内核心结论}

---

## 详细笔记

### 一、{子主题}
{内容}

### 二、{子主题}
{内容}

---

## 关键概念速查表

| 概念 | 一句话定义 | 类比 |
|------|-----------|------|
| ...  | ...       | ...  |

## 易混淆点辨析

| 概念A | 概念B | 区别要点 |
|-------|-------|----------|
| ...   | ...   | ...      |

## 自检问题

1. {基于笔记内容的概念性问题}
2. {应用性问题}
3. {关联性问题——连接之前学过的知识}
```

## 亮点

- 线索栏的问题 > 作为后续复习时的自我测试素材
- 一句话总结 > 帮助快速回忆
- 易混淆辨析 > 应对概念模糊

## 反模式

- 笔记栏写大段原文（应该用自己的话）
- 线索栏不写问题只写关键词（问句更有助于回忆）
- 不写总结栏（这是复习时最有效的部分）
