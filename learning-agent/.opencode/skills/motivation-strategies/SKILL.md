# 学习动机策略

**理论基础**：
- 自我决定理论 (Ryan & Deci, 2000)：三大基本心理需求——自主性、胜任感、关联感
- SMART 目标框架 (Doran, 1981)
- 自我效能理论 (Bandura, 1977)

## 策略一：SMART 目标设定

每个学习目标应满足：

| 维度 | 要求 | 示例 |
|------|------|------|
| **S**pecific 具体 | 说清楚学什么 | "学完微积分第3章的导数和微分" |
| **M**easurable 可衡量 | 有事后的检测方式 | "能独立做5道求导题" |
| **A**chievable 可达成 | 在现有水平上略超一点 | 不是"一天学会微积分" |
| **R**elevant 相关 | 与用户的目标相关 | 为考试 → 侧重考点 |
| **T**ime-bound 有时限 | 有明确的完成时间 | "本周五前完成" |

## 策略二：自主性支持

- 给用户选择权："你想先学 X 还是先学 Y"
- 解释"为什么学这个"——建立知识点的意义感
- 鼓励用户提出自己的学习节奏偏好

## 策略三：胜任感反馈

- 成功后即时肯定：具体赞扬（"你用极限定义推导了导数，概念理解很扎实"）而非泛泛"很好"
- 遇到困难时：强调进步而非差距（"你比上次多理解了一个关键步骤"）
- 避免与其他人的比较

## 策略四：激励节点

在计划中嵌入动力节点：

```markdown
🏁 里程碑 1：完成第 1-3 章 → 🎯 阶段进度 30%
   ✓ 奖励："你已经完成了最难的入门部分！后面会越来越顺"

🏁 里程碑 2：完成第 4-6 章 → 🎯 阶段进度 60%
   ✓ 奖励："超过一半了！回顾一下，你比两周前多掌握了多少？"

🏁 里程碑 3：完成所有新课 → 🎯 阶段进度 80%
   ✓ 奖励："进入复习阶段！这才是真正的升华"

🏁 里程碑 4：完成总复习 → 🎯 阶段进度 100%
   ✓ 奖励："恭喜！你已经从'了解'走到'掌握'了"
```

## 策略五：克服拖延

- 番茄钟降低启动成本（"只学 25 分钟"比"学一下午"更容易开始）
- 拆分庞大任务："学完线性代数" → "今天只学矩阵乘法"
- 记录已完成项（学习历史中看得到进度本身就是激励）

## 在计划中嵌入

在输出学习计划时，自动应用以上策略，而非只用任务列表。

## 反模式

- 目标太宏大无法衡量
- 只给进度不给意义（"学到第 5 章了"不如"你已掌握了解决实际问题的工具"）
- 频率过高（每周 1-2 个里程碑即可，太多变成干扰）
