## 阶段一：突破卡点 + 巩固加强（优先级最高）

### 1.1 P1080 国王游戏（当前卡住）
- **状态**：sort 比较函数为空，贪心排序策略未写出
- **核心难点**：需要证明按照 `a_i * b_i` 排序的贪心策略正确性
- **学习路径**：
  1. 理解"邻项交换"证明法（贪心排序的经典证明技巧）
  2. 手写比较函数并验证
  3. 注意高精度乘除法的正确性
- **建议耗时**：2-3 次集中练习

### 1.2 线段树 lazy 标记深化
- **状态**：已从零实现字符映射线段树，但可读性和理解可进一步提升
- **学习路径**：
  1. 复习线段树基本操作（build/update/query）
  2. 深入理解 lazy 标记的下传时机和合并逻辑
  3. 练习 1-2 道经典线段树题目（如区间加+区间求和）
- **建议耗时**：2 次集中练习

## 阶段二：数论基础（重大空白区）

### 2.1 基础数论工具包
- GCD / LCM（欧几里得算法与扩展欧几里得）
- 素数筛（埃氏筛、线性筛）
- 快速幂（二分幂、矩阵快速幂）
- 模运算（逆元、费马小定理）
- 组合数计算（阶乘预处理+逆元）

### 2.2 推荐学习顺序
1. GCD/LCM → 扩展欧几里得
2. 素数筛 → 快速幂
3. 模运算与逆元 → 组合数

### 2.3 练习建议
- 洛谷 P3383（线性筛素数）
- 洛谷 P1226（快速幂模板）
- 洛谷 P3811（乘法逆元）
- 洛谷 P3807（组合数模版）

## 学习节奏建议

```
每周 3-4 次集中练习，每次 1-2 小时：
- 2 次用于当前阶段主线内容（新知识学习 + 模板题）
```
