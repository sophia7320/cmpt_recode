- 目标：竞赛/面试级别全面掌握，补齐知识空白

## 各科目计划

### 算法与数据结构
> 详见 [算法与数据结构详细计划](./plans/算法与数据结构-计划.md)

**总体策略**：
1. **巩固强项**：贪心、DP、图论最短路——通过变种题加深
2. **突破卡点**：优先解决 P1080 国王游戏，理解贪心排序策略的证明
3. **补齐空白**：按优先级——数论基础 → 字符串算法 → 网络流 → 树高级算法
4. **强化薄弱**：线段树 lazy 标记 + 树形DP + 状压DP

**学习顺序建议**：
```
阶段一（当前）：突破 P1080 国王游戏 + 巩固线段树
阶段二：数论基础（GCD/LCM、素数筛、快速幂、模运算）
阶段三：字符串算法（KMP、字符串哈希深入）
阶段四：网络流入门（最大流基础）
阶段五：树高级算法（LCA、树形DP深化）
```
